# Praktikum 0x2 Arsikom 

- [PDF Instruksi](<./Instruksi Pra-Praktikum dan Praktikum 0x2 IF1230 (2026).pdf>)

Pada praktikum ini, kalian diminta melakukan eksplorasi cara kerja program dengan membaca dan memahami kode assembly.
Tugas kalian mencari input benar yang diminta oleh berbagai fungsi dalam file ELF (Executable and Linkable Format) yang diberikan.
Kalian diharapkan menggunakan GDB (GNU Debugger) dan diperbolehkan (bahkan, dianjurkan) menggunakan GDB PEDA atau PWNDBG (ini yang lebih modern) sebagai alat analisis file ELF. 

Untuk menggunakan GDB, kalian perlu menggunakan Linux. Terdapat beberapa opsi yang dianjurkan:
- Menggunakan Google Collab.
- Menggunakan Windows Subsystem for Linux (WSL).
- Menggunakan OS Linux sesungguhnya, misalnya dengan Dual Boot.
